Matt Garmon - 2017 


A simulation of the card game war.



Default options:

num_tests = 10000

num_decks = 1 
					
num_sacrifices = 3
					
shuffled_aquisition = y
					
negotiable_sacrifice = y



Command line execution options:



./war		
./war num_tests
./war num_tests num_decks num_sacrifices shuffled_aquisition negotiable_sacrifice
 

(use y or n for shuffled_aquisition negotiable_sacrifice)
